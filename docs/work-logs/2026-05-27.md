- **작업 내용**: 모바일에서 손은 인식되나 소리가 나지 않던 자동 재생 차단 문제 해결.

- **구체 사안**:

  - 매 프레임 손 인식 좌표 갱신 루프(`updateRightHand`) 내부에서 모바일 백그라운드 안전 차단망으로 `ctx.state === 'suspended'`인 상태가 발견될 시 즉각 자동 잠금 해제(Auto-Resume) 기능 결합.

### 3. 볼륨 제어 Span 임계치 반응 마진 대폭 완화

- **작업 내용**: 스마트폰 카메라와 손 간의 거리가 멀어지면 볼륨이 자동으로 소거(Mute)되는 임계치 한계를 보정하여 연주 반응성 대폭 강화.
- **수정 파일**: [useThereminAudio.ts](file:///c:/Users/Dolveul/Desktop/Project/air%20theremin/src/hooks/useThereminAudio.ts)
- **구체 사안**:
  - 최소 소리 반응 음소거 범위를 `calibratedSpan * 0.4`에서 `calibratedSpan * 0.2`로 2배 하향.
  - 최대 볼륨 매핑 상한선을 `calibratedSpan * 1.5`에서 `calibratedSpan * 1.2`로 부드럽게 매칭하여 근/원거리 모든 스케일에서 원활한 연주 볼륨 확보.

### 4. 리액트 비동기 stale closure 방어

- **작업 내용**: MediaPipe 백그라운드 콜백에서 리액트의 `isAudioStarted`, `isCalibrating`, `isSwapped` 상태값을 읽을 때, 초기 `false` 고정 스태일 버그로 인해 렌더 파이프라인이 멈추고 게이지가 0에 고정되던 구조적 버그 해결.
- **수정 파일**: [HomePage.tsx](file:///c:/Users/Dolveul/Desktop/Project/air%20theremin/src/pages/HomePage.tsx)
- **구체 사안**:
  - 최신 렌더 상태를 실시간 스냅샷으로 확보하는 Ref 클론(`isAudioStartedRef`, `isCalibratingRef`, `isSwappedRef`) 구축.
  - 실시간 리액트 상태 동기화 `useEffect` 훅 연동 및 MediaPipe 콜백 내에서 Ref 참조 방식으로 전면 수정하여 원천 해결.

### 5. 다중 손 교차 간섭 방지 물리 정렬 시스템 구현

- **작업 내용**: 왼손을 크게 움직여서 중앙에 가까워질 때 오른손의 음정이 순간적으로 튀거나 교차되던 오류 완벽 해결.
- **수정 파일**: [HomePage.tsx](file:///c:/Users/Dolveul/Desktop/Project/air%20theremin/src/pages/HomePage.tsx)
- **구체 사안**:
  - MediaPipe Hands가 무작위 순서로 손 배열을 담는 한계를 극복하기 위해, 감지된 모든 랜드마크를 `wristX` (미러링된 X좌표) 기준 오름차순으로 사전 정렬.
  - 2개의 손이 동시 인식될 경우 화면상의 왼쪽 손은 강제로 왼손(이펙터), 오른쪽 손은 오른손(연주 전용)으로 매핑하여 스왑 간섭 0% 보장.
  - 1개의 손만 인식될 경우 절대 영역 분할 방식으로 부드럽게 롤백되는 예외 안전장치 탑재.

### 6. 오른손 손가락 끝(Fingertips) 주파수 매핑 및 주먹 쥐기(Openness) 볼륨 제어 도입

- **작업 내용**: 손바닥이나 손목의 뒤틀림으로 인해 음정이 튀는 현상을 방지하고, 오무려 쥐는 형태(주먹)로 음량을 극대화/소거할 수 있도록 감도 개편.
- **수정 파일**: [HomePage.tsx](file:///c:/Users/Dolveul/Desktop/Project/air%20theremin/src/pages/HomePage.tsx)
- **구체 사안**:
  - 손 전체 랜드마크 스캔 대신 **5개 손가락 끝(Thumb, Index, Middle, Ring, Pinky Tip)의 최우측값**만 추적하도록 변경하여 손날/손바닥 오류 완벽 방지.
  - 기존 엄지-새끼 끝간 거리 외에, 네 손가락 끝과 마디(MCP) 사이의 수축 거리를 수치화한 **하이브리드 펼침(Openness) 필터**를 탑재하여 주먹을 자연스럽게 쥘 때 볼륨이 0으로 완벽 음소거되도록 구현.

### 7. 왼손 펼침/오므림(Openness) 이펙터 전환 및 실시간 Cyan 입자 비주얼 트래커 구축

- **작업 내용**: 왼손 Y축을 화면 높낮이 대신 손의 펼침 강도로 제어하도록 교체하고, 왼손 위치를 실시간 확인 가능한 시각 효과 적용.
- **수정 파일**: [HomePage.tsx](file:///c:/Users/Dolveul/Desktop/Project/air%20theremin/src/pages/HomePage.tsx)
- **구체 사안**:
  - 기존 왼손 Y축 화면 좌표 대신, 오른손과 대칭되는 **왼손 펼침/오므림 정도(Openness)**를 계산하여 필터 컷오프(Lowpass Timbre)를 매핑.
  - 왼손 손목의 미러링 2D 실시간 좌표를 `rawLeftHandRef`에 영사하고, Canvas의 파티클 렌더 시스템에 **안정적인 Cyan(청록색) 입자 트레일 효과**를 결합하여 왼손 감지 및 제어 상태를 눈으로 실시간 확인 가능하도록 완벽 가이드 제공.
